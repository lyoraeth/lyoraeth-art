# Security Policy

## Scope

This policy covers the source code in this repository and the live site at [lyoraeth.art](https://lyoraeth.art), including its API endpoints.

Out of scope: third-party services (Sanity, Resend, Cloudflare Turnstile) — report those to their respective vendors.

## Reporting a Vulnerability

**Please do not open a public GitHub issue for security vulnerabilities.**

Report privately via email: [lyoraeth@gmail.com](mailto:lyoraeth@gmail.com)

You can encrypt your message using the PGP key at [lyoraeth.art/pgp-key.txt](https://lyoraeth.art/pgp-key.txt).

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

## Response

I aim to acknowledge reports within 72 hours and resolve confirmed issues within 14 days. I'll keep you updated on progress and credit you in the fix if you'd like.
